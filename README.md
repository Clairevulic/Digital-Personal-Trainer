# Digital-Personal-Trainer

# This project was developed on Visual Design Code
# This code datafies the decision-making processes employed by personal trainers
# The scope of Digital Personal Trainer is to translate the logic behind trainers' reasoning and recommendation into code

# The project will:
# Greet the user
# Ask for users name
# Asks the user for their goal
# Asks the user for a focus
# Recommend the user an exercise
# The user is then given the opportunity to select a new goal, select a new focus from their original goal or exit

# Exercises will not be recommended twice

# How to run:
# This program requires Visual Studio Code
# This code should be copied or downloaded 
# This code should be pasted or placed in Visual Studio Code
# Run the program
# Follow the onscreen prompts

# The program is interactive: it will prompt the user to answer questions that affect the following questions
